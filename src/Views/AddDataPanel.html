Add your own data

Use this panel to add data to the map from your local system or from elsewhere on the web. Data added in this way is not saved or made visible to others unless you explicitly share it by using the Share panel.

Supported data formats:

GeoJSON, KML, KMZ, CSV (with lat/lon columns), CZML, and more.

Enter a web link for a single data file or for a WMS or WFS service:

Or, browse to select a local data file to upload

TIP: you can also drag data files directly onto the map.
